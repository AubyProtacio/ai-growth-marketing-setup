# AI-Assisted Growth Marketing Workflow Setup

## Project Overview

This repository documents the setup process completed as part of the 100Hires Junior Growth Marketing Specialist application task.

The objective was to configure an AI-assisted workflow environment, document the setup process, troubleshoot technical challenges independently, and demonstrate clear and structured technical documentation.

---

## Tools Installed

* Cursor IDE
* GitHub account and public repository setup
* GitHub integration with Cursor

---

## Steps Completed

### 1. Installed Cursor IDE

Downloaded and installed Cursor successfully.

### 2. Connected GitHub to Cursor

Authorized GitHub access and completed account integration. This was completed by signing in with an existing GitHub account to enable seamless integration.

### 3. Attempted Extension Installation

Attempted to install Claude Code and Codex; however, the add-ons were not available in the marketplace.

To verify availability, additional related terms were searched:

* Claude
* Anthropic
* OpenAI

### 4. Created Public GitHub Repository

Created a public repository to document the setup process and maintain transparent project documentation.

### 5. Created README Documentation

Documented the installation workflow, troubleshooting process, observations, and outcomes.

---

## Issues Encountered

### Extension Marketplace Availability

The requested extensions (“Claude Code” and “Codex”) were not discoverable within the current Cursor extension marketplace.

---

## Troubleshooting Steps Taken

To resolve this issue, I:

* Searched exact extension names
* Searched related keywords
* Verified Cursor installation
* Confirmed GitHub integration was active
* Reviewed extension search results multiple times

---

## Resolution

The extension availability appears to differ from the provided instructions, which may indicate marketplace changes, version differences, or environment-specific availability.

This was documented transparently while continuing with the remaining required steps.

---

## Key Learnings

This task reinforced the importance of:

* Careful review and accurate execution of provided instructions
* Independent troubleshooting
* Clear technical documentation
* Adapting to evolving tool environments for workflow efficiency
* Verifying implementation instructions across available AI tools and digital platforms
* Maintaining transparency when encountering technical discrepancies
